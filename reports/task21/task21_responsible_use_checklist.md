# Task 21 Responsible Use Checklist

This checklist defines how the sarcasm detection system should be presented and used responsibly.

- [x] Use the model for academic demonstration and exploratory analysis only.
- [x] Do not use the model to automatically remove, flag, penalize, or judge users.
- [x] Show the prediction confidence and communicate uncertainty.
- [x] Avoid exposing raw Reddit text samples in public reports or repositories.
- [x] Keep human review in the loop for any interpretation of model outputs.
- [x] Disclose dataset, preprocessing, evaluation, and model limitations clearly.

## Public Demo Requirement

If the application is hosted publicly, the public version should focus on inference and report presentation only. Heavy task execution or model training should remain local unless the hosting environment is explicitly configured for it.
