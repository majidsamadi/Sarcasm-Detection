Model experiment design completed. Four controlled experiments were defined: BERTweet with Version A, BERTweet with Version B, RoBERTa with Version A, and RoBERTa with Version B. All experiments use the same stratified train/validation/test split, text-only input, max length 128, and the same evaluation metrics: accuracy, precision, recall, F1-score, and confusion matrix. The final implementation will deploy only the best-performing model based mainly on F1-score and error analysis.
